### 주요 기능
- 크래프톤 정글 교육생들의 TIL·WIL을 아카이브하는 크롬 확장 프로그램
- 교육생들이 작성한 글을 모아 한 곳에서 열람 및 검색 가능
- 개인별 학습 기록을 저장하고 공유할 수 있는 기능 제공

### Tech Stack
- Frontend: React, Shadcn/ui, Tailwind CSS  
- Backend: Python, Flask, MongoDB

### Role
> 프로젝트 기획 및 개발

#### 프론트엔드 개발
- React 기반 UI 구현 및 Shadcn/ui, Tailwind CSS를 활용한 확장 프로그램 인터페이스 개발
- 아카이브 검색 및 필터링 기능을 제공해 사용자의 접근성 강화

#### 백엔드 개발
- Flask REST API 서버 설계 및 MongoDB 연동
- TIL/WIL 데이터의 CRUD 기능 및 사용자 인증 로직 구현

#### 프로젝트 성과
- 크롬 브라우저 환경에서 간편하게 학습 기록을 모아보고 공유할 수 있는 도구 제공
- 정글러 커뮤니티의 학습 기록 관리와 지식 공유를 촉진
