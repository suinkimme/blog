### 주요 기능
#### AI 모의면접
- 이력서를 기반으로 맞춤 질문을 생성하고 실제 면접처럼 실시간 모의면접 진행  
- 실시간 영상/음성 분석을 통해 답변 태도·내용 평가 및 결과 리포트 제공  

#### 라이브 커리어 멘토링
- 1:1 화상 연결을 통한 실시간 이력서 코칭  
- 펜, 형광펜, 스티커 메모 등 다양한 협업 도구 제공  
- 멘토 신청, 관리, 피드백, 리플레이 기능 제공  

### Tech Stack
- **Frontend**: Next.js, Shadcn/ui, Tailwind CSS  
- **Backend**: NestJS, MySQL  
- **실시간 통신**: Socket.IO, WebRTC, Yjs (CRDT)

### Role
> 1:1 라이브 커리어 멘토링 및 결제 연동 기능 개발

#### 1:1 라이브 커리어 멘토링
- 펜, 형광펜, 스티커 메모, 녹음 등 **실시간 협업 도구** 제공  
- 구직자와 멘토가 동시에 문서를 작성·피드백할 수 있는 환경 구현  
- 기존 Lock 제어 방식 한계를 개선하고 **Yjs 기반 CRDT 동기화**를 적용해 **안정적인 동시 편집** 지원  

#### 멘토링 상품 관리 및 결제 연동
- 멘토링 서비스 **유료화를 위한 상품 등록·결제·운영 관리 화면 구현**  
- **PortOne 결제 모듈** 연동으로 카드·간편결제 플로우 지원  
- 관리자 전용 화면에서 상품 CRUD 기능 제공  
- 결제 완료 시 **웹훅 응답 자동 처리**로 주문 상태가 자동 반영되어 운영자가 개입하지 않아도 관리 가능
