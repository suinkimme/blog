### 주요 기능
- 크래프톤 정글 교육생 대상 회의실 예약 PWA 앱
- 시간대별 팀 단위 예약 제한(각 팀당 1명만 예약 가능)
- GitHub 로그인 연동으로 교육생 인증 및 접근 제어
- 공정한 회의실 사용을 지원하여 운영 문제 해소

### Tech Stack (FE)
- React, PWA, Shadcn/ui, Tailwind CSS

### Role
> 프론트엔드 개발 전반 담당

#### 회의실 예약 기능
- 시간대별 예약 제한 로직을 구현해 중복 예약 방지
- 예약 현황을 실시간으로 표시해 사용자 간 충돌 최소화

#### 사용자 인증
- GitHub OAuth를 활용한 로그인 기능 도입으로 사용자 인증 및 접근 제어

#### 빠른 개발 및 배포
- 약 3일 만에 MVP를 개발해 배포
- 배포 직후 회의실 관련 불공정 문제 해소 및 사용자 만족도 향상
