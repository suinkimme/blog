### 주요 기능
- 포인트 적립·사용, 상품 구매, 선물 및 공유 기능 제공
- QR 기반 오프라인 결제 지원
- 안정적인 포인트 결제 및 인증 처리

### Tech Stack
- Frontend: React Native, Android, Firebase Dynamic Links
- Backend: Express.js, MySQL


### Role
> 프론트엔드 및 백엔드 API 개발을 포함한 풀스택 개발 주도

#### QR 기반 오프라인 결제 구현
- 매 스캔마다 새로운 QR을 생성하는 일회성 결제 구조로 보안성 강화
- Android에서 FLAG_SECURE를 동적으로 제어하는 Native 모듈을 구현해 민감 정보 노출 방지

#### 포인트 결제 연동
- KG이니시스 제휴 심사 및 기술 검수 진행 후 PortOne 라이브러리를 React Native 환경에 적용
- 포인트 결제 및 인증 절차를 안정적으로 처리할 수 있는 구조 구축
