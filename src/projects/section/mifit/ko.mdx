### 주요 기능
- 직원들의 헬스 이용을 안전하게 인증하여 매월 헬스비 지원을 받을 수 있도록 지원
- 카메라 기능으로 시계 촬영을 통한 헬스 인증
- GPS 기반 위치 검증으로 등록된 헬스장 주변에서만 인증 가능
- 관리자용 대시보드에서 참여 현황을 시각화해 지원비 지급 대상자 확인

### Tech Stack (FE)
- React Native

### Role
> 프로젝트 매니징 및 프론트엔드 전반 개발·디자인

#### 인증 프로세스 구현
- 카메라 API를 활용하여 운동 참여 인증을 위한 사진 촬영 기능 개발
- GPS 기반 위치 검증 로직을 적용해 허위 인증 방지

#### 관리자용 대시보드 개발
- 참여 현황 시각화를 통해 지원비 지급 대상자를 직관적으로 확인 가능하도록 구현

#### UI/UX 설계 및 디자인
- 디자이너 협업 없이 직접 화면 설계 및 UX 개선
- 색상, 타이포그래피, 간격 등 공통 속성을 디자인 토큰으로 정의해 일관된 UI 구현

#### 배포 및 운영
- 비공개 배포 방식으로 특정 사용자(링크 보유자, 초대 대상자)만 다운로드 가능하도록 설정
- 테스트 및 사내 배포 환경을 분리해 안정적인 앱 운영 지원
