### 주요 기능
- 네이버/다음 검색 데이터를 기반으로 키워드 검색량, 자동완성, 연관검색 제공
- 쇼핑·블로그 등 섹션별 노출 정보 분석

### Tech Stack (FE)
- React Native, TanStack Query

### Role
> 프론트엔드 개발 및 프로젝트 매니지먼트

#### 웹뷰 기반 앱의 네이티브 마이그레이션
- 기존 웹앱을 React Native로 재구현하여 성능 및 사용자 경험 개선

#### UI/UX 직접 설계 및 구현
- 디자이너 협업 없이 Figma를 활용해 핵심 UI 컴포넌트와 화면 구성 직접 설계
- UX 개선을 위한 컴포넌트 구조 및 인터랙션 정의

#### Skeleton UI 및 애니메이션 구현
- React Native Linear Gradient와 Animated API를 활용해 Shimmer 애니메이션 제작
- 콘텐츠 유형별 Skeleton UI 컴포넌트 개발로 로딩 경험 개선
